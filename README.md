# Week 4 - Task 2: Machine Learning Workflow

## Objective
This project demonstrates a complete machine learning workflow using a public dataset. It includes data preprocessing, exploratory data analysis (EDA), model training, evaluation, and comparison.

## Dataset
Dataset: Iris Dataset
Source: https://archive.ics.uci.edu/ml/datasets/iris

## Project Workflow
1. Data Collection
2. Data Cleaning and Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Model Comparison
9. Business Insights

## Models Used
- Logistic Regression
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Libraries
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
